# SimpleTaskBoard-PHP
 A lightweight PHP web application that
  allows users to manage a simple task list.
   The project is fully containerized using
    Docker for easy and consistent setup.
     ## Tech Stack 
     - PHP 8.2 
     - Apache 
     - Docker 
     ## How to Run with Docker
      ### Build the image
       ```bash
        docker build -t simple-task-board .
